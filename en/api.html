new Monitor([url], api_key, monitor_key)

Creates a Monitor instance capable of communicate with a Wireless Monitor server located at the url you specify.

Param Type Default Description
[url] String http://localhost:8000 URL of the server
api_key String This value specifies the user. Must be in UUID format.
monitor_key String This value specifies the device. Must be in UUID format.
